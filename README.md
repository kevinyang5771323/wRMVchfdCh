## 项目概述

通讯录记账系统是一个基于Python和PyQt5开发的桌面应用程序，使用MySQL数据库进行数据存储，具有良好的可定制性。

## 技术栈

- 主要编程语言：Python
- 用户界面框架：PyQt5
- 数据库操作：PyMySQL
- 数据库系统：MySQL 8.0及以上

- 管理用户信息

- 普通用户：使用通讯录和记账功能
- 管理员：管理用户信息和数据

- 操作系统：支持PyQt5的操作系统
- Python环境：建议使用Python 3.x版本
- 数据库环境：MySQL 8.0及以上

1. 安装Python环境
2. 安装PyQt5库
3. 安装PyMySQL库
